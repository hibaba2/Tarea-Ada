## Como Usar

Generar archivo `data.txt` con 10M datos aleatorios

    $ make generar

Ejecutar algoritmos para 10M datos

    $ make

Ejecutar algoritmos para 10M datos y exportar `logs.txt`

    $ make exportar

Testear algoritmos solo con 10K datos

    $ make test
